# Processed Data (data/processed/)

This directory contains data that has been cleaned and formatted for analysis.

## Subdirectories

| Directory | Description |
| :--- | :--- |
| `detections/` | Cleaned detection histories and deployment records. |
| `occurrence/` | Grid-level spatial covariates. |
| `results/` | Outputs from occupancy models (JAGS, spOccupancy). |
