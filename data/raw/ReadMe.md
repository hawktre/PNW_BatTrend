# Raw Data (data/raw/)

This directory contains original, unmodified data files. **Files in this directory should not be modified directly.**

## Files Overview

| File | Description | Source |
| :--- | :--- | :--- |
| `SpeciesNatHistMatrix.xlsx` | Matrix of bat species natural history traits. | Project team |

## Subdirectories

| Directory | Description |
| :--- | :--- |
| `batgrid/` | NABat master sample 10km grid shapefiles. | NABat |
| `covariates/` | Spatial and climate covariate data (LandFire, Daymet). | USGS/NASA |
| `database/` | MS Access databases containing the primary bat monitoring data. | PNW Bat Hub |
| `tables/` | CSV exports of the MS Access database tables. | Generated by `00_ExportTables.sh` |
